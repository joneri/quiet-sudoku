# DI-001: Playable floating Sudoku window

## Outcome

A runnable macOS app opens directly into a floating always-on-top Sudoku board with no toolbar, visible title, or in-window buttons.

## Acceptance

- App launches as a foreground macOS app bundle.
- Main window uses a glass/material feel and floats above normal windows.
- Sudoku board fills the content area from edge to edge.
- User can select empty cells and enter digits 1-9.
- User can clear editable cells with delete/backspace/0.
- Given puzzle values are protected from edits.
- Build/run entrypoint exists at `script/build_and_run.sh`.
